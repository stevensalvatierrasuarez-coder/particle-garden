# Particle Garden 🎨

Un pequeño lienzo interactivo de partículas, hecho en HTML5 Canvas puro (sin dependencias).

## Cómo usarlo

Abre `index.html` en cualquier navegador moderno:

```bash
open index.html
```

O sírvelo con cualquier servidor estático, por ejemplo:

```bash
python3 -m http.server
```

y visita `http://localhost:8000`.

## Controles

- **Clic y arrastra** sobre el lienzo para emitir partículas.
- **Modo**: alterna entre `Fuente` (emisión continua) y `Explosión` (ráfaga puntual).
- **Gravedad**: ajusta qué tanto caen las partículas.
- **Cantidad**: número de partículas generadas por interacción.
- **Color**: elige una paleta fija o `Arcoíris` para colores aleatorios.
- **Limpiar**: borra todas las partículas del lienzo.
